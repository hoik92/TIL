# 02. 개발 환경 설정

## 2-1. JDK 설치

* JSP 및 Servlet은 Java를 기본언어로 사용한다.
* 그러므로 JDK(Java Development Ket)를 설치한다.

1. [https://oracle.com/kr/index.html](https://oracle.com/kr/index.html)에 접속
2. JDK 설치(JDK 8u221)
